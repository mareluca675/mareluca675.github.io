FAQ

What AI model is used?

A pre‑trained YOLOv8 model (e.g., yolov8x.pt) for real‑time bear detection.

How does detection work?

Each frame is resized to ~640×640 and processed once by YOLO. We keep detections where the class is bear and confidence exceeds the threshold, then draw boxes and scores.

How do I run it?

pip install -r requirements.txt
python setup_script.py
python bear_detection.py                 # webcam
python bear_detection.py --source video.mp4   # video file

What options are available?

--model yolov8x.pt
--confidence 0.3
--source 1
--no-display
--width 1080 --height 720

Minimum & recommended specs?

  • Minimum: Python 3.8+, 4 GB RAM, CPU, webcam/video file
  • Recommended: Python 3.10+, 8 GB+ RAM, NVIDIA GPU with CUDA, 1080p+ camera

Is there a Python API?

Yes — import BearDetector and use process_video_stream or detect_bears for single frames.
