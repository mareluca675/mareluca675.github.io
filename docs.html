AI Detection System

The core detection system uses YOLOv8 for real-time bear identification in video streams. Designed for edge deployment in remote locations with minimal infrastructure.

Installation

# Clone the repository
git clone https://github.com/mareluca675/bear-detection-system.git
cd bear-detection-system

# Install dependencies
pip install -r requirements.txt

# Download YOLOv8 model
python setup_script.py

Basic Usage

# Run with webcam
python bear_detection.py

# Run with video file
python bear_detection.py --source path/to/video.mp4

# Run with custom camera
python bear_detection.py --source 1

# Run headless (no display)
python bear_detection.py --no-display

Command Line Options

Option Description Default
--source Video source (0, 1, 2 for cameras or file path) 0
--model YOLO model file yolov8x.pt
--confidence Detection confidence threshold (0-1) 0.5
--no-display Run without video display False
--no-alerts Disable Firebase alerts False
--firebase-url Firebase app URL From config
--camera-lat Camera latitude 45.5231
--camera-lng Camera longitude -122.6765
--camera-id Camera identifier camera_01

Threat Level Classification

Detections are automatically classified into three threat levels:

  • HIGH - Confidence ≥ 90% (Red alert)
  • MEDIUM - Confidence ≥ 70% (Orange alert)
  • LOW - Confidence ≥ 50% (Green alert)

Alert Payload Structure

{
  "latitude": 45.5231,
  "longitude": -122.6765,
  "confidence": 0.85,
  "timestamp": 1701439200000,
  "source": "camera_01",

  "bbox": [120, 80, 380, 420],
  "detectionTime": "2024-01-15T10:30:00",

Mobile Application

The BearAware mobile app provides real-time bear alerts, interactive maps, and educational resources for hikers and outdoor enthusiasts.

Key Features

Real-Time Alerts

Instant push notifications when bears are detected in your area

Interactive Maps

Live bear locations and danger zones overlaid on detailed topographic maps

User Reporting

Submit bear sightings and contribute to community awareness

Safety Guidelines

Educational content on bear behavior and encounter prevention

Setup & Development

# Clone the repository
git clone https://github.com/mareluca675/bear-detection-app.git
cd bear-detection-app

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env.local
# Add your Firebase configuration

# Run development server
npm run dev

# Build for production
npm run build

Firebase Configuration

Create a .env.local file with your Firebase credentials:

NEXT_PUBLIC_FIREBASE_API_KEY=your_api_key
NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN=your_auth_domain
NEXT_PUBLIC_FIREBASE_PROJECT_ID=your_project_id
NEXT_PUBLIC_FIREBASE_STORAGE_BUCKET=your_storage_bucket
NEXT_PUBLIC_FIREBASE_MESSAGING_SENDER_ID=your_sender_id
NEXT_PUBLIC_FIREBASE_APP_ID=your_app_id

Deployment Guide

Edge Device Deployment

Deploy the AI detection system on edge devices in remote locations:

  1. Hardware Requirements
    • NVIDIA Jetson Nano/Xavier (recommended) or Raspberry Pi 4
    • USB camera or IP camera
    • 4G/LTE modem for connectivity
    • Weather-proof enclosure
  2. Software Setup
    • Install Ubuntu 20.04 or JetPack (for Jetson devices)
    • Install CUDA drivers (if using GPU)
    • Clone and install bear detection system
    • Configure Firebase credentials
  3. Configuration
    • Set camera location coordinates
    • Adjust confidence thresholds
    • Configure alert cooldown period
    • Test connectivity to Firebase

Mobile App Deployment

# Build the application
npm run build

# Deploy to Firebase Hosting
firebase deploy --only hosting
