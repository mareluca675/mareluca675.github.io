Real‑time Bear Detection

A minimal, focused implementation that uses a pre‑trained YOLOv8 model to detect bears in video streams.

Get Started View on GitHub

How it works

The system captures frames from a camera or video file, resizes them to 640×640, runs YOLO once per frame, keeps only detections where the class is bear above a confidence threshold, and draws bounding boxes with confidence scores.

  1. Input: Camera or video file
  2. Detection: YOLO processes each frame
  3. Filtering: Keep bear above threshold
  4. Output: Boxes + confidence overlaid

Camera/Video → Frame Extraction → YOLO Model → Object Detection → Bear Filter → Display

What’s inside

  • Python
  • YOLOv8 model weights (e.g., yolov8x.pt)
  • OpenCV for capture and display

Run in 3 steps

pip install -r requirements.txt
python setup_script.py
python bear_detection.py            # webcam
python bear_detection.py --source video.mp4   # video file

Handy options

--model yolov8x.pt
--confidence 0.3
--source 1
--no-display
--width 1080 --height 720
