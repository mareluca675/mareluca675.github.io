Fundament Științific Solid

BearAware este construit pe cercetare multidisciplinară riguroasă, combinând fizică acustică, chimie organică, inteligență artificială și etologie.

🔊 Bioacustică & Fizică

  • Frecvență Ultrasonică: 21 kHz (±10%) — optimizată pentru sensibilitatea auditivă maximă a urșilor
  • Presiune Acustică: 60-100 dB — suficient pentru descurajare, sub pragul de deteriorare auditivă
  • Propagare Sinusoidală: Unde pure pentru penetrare maximă în teren montan
  • Modelare Matematică: Legea Inversului Pătrat (I₁/I₂ = r₂²/r₁²) pentru calculul razei de siguranță
  • Alternanță Temporală: 20s ultrasunete + 10s infrasunete — previne habituarea și optimizează eficiența

🧪 Chimie & Repelent Natural

  • Mentol (C₁₀H₂₀O): 30-55% — efect răcoritor și respingător olfactiv
  • Mentonă (C₁₀H₁₆O): 10-30% — intensifică aromă și efectul terapeutic
  • Eucaliptol (C₁₀H₁₈O): 3-15% — notă camforată, antiseptic natural
  • Metil-Acetat: Agent volatilizant pentru dispersie rapidă
  • Eficiență Estimată: ~66% (model probabilistic bazat pe studii comparative)

🤖 Inteligență Artificială

  • Model YOLOv8: Rețea neuronală convoluțională (CNN) pentru detecție în timp real
  • Training Dataset: ~700 imagini annotate cu urși + clase negative (oameni, animale domestice)
  • Validare Multi-Spectrală: RGB + infraroșu pentru detecție nocturnă
  • Activare Condiționată: Repelenți activați doar la confirmare vizuală — zero alarme false
  • Logging Automat: GPS + timestamp pentru analiză etologică

📚 Cercetare Aplicată

  • Studii de Referință: Miller (1978) — ultrasunete pe urși polari (69% succes)
  • Bioacustică Avansată: Vocalize agresive sintetizate — relevanță biologică pentru Ursus arctos
  • Studiu Tibetan (2025): AI + spray capsaicină (75% eficiență) — adaptat cu mentol natural
  • Colaborare Viitoare: Sanctuare urși + Salvamont pentru validare experimentală

How it works

The system captures frames from a camera or video file, resizes them to 640×640, runs YOLO once per frame, keeps only detections where the class is bear above a confidence threshold, and draws bounding boxes with confidence scores.

  1. Input: Camera or video file
  2. Detection: YOLO processes each frame
  3. Filtering: Keep bear above threshold
  4. Output: Boxes + confidence overlaid

Camera/Video → Frame Extraction → YOLO Model → Object Detection → Bear Filter → Display

Technology Stack

  • Python & YOLOv8 (AI System)
  • Mobile frameworks (Detection App)
  • OpenCV for image processing
  • Real-time computer vision

pip install -r requirements.txt
