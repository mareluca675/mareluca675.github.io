Studies

Attach PDFs, DOCX files, and other references. To add items, commit your files into the /studies folder of this site and list them in /studies/studies.json.

  1. Drop your documents into /studies/ (e.g., studies/bear-paper.pdf).
  2. Edit studies/studies.json and add a new entry with the filename and metadata.
  3. Commit and push — this page will auto-render the list.

Example JSON entry

{
  "title": "Bear Activity and Habitat 2025",
  "filename": "bear-activity-2025.pdf",
  "authors": "A. Researcher, B. Scientist",
  "year": 2025,
  "tags": ["behavior","habitat"]
}

Note: PDFs can be previewed inline below each card. DOCX and other formats will show a download link (browser preview support varies).
